# Getting and Cleaning Data - Course Project

## Data Description
http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones

## Data Download
https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 

The data is broken down into test and train data and has 561 features.  There is also a subject and activity data set for each observation which is stored in different data files.

The data as presented did not have easily readable names.

In order for the script to run, the data set must be downloaded and unzipped in the R working directory
